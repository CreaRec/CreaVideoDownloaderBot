# Telegram Video Downloader

Telegram Video Downloader is a private Node.js service that listens to your Telegram bot and downloads videos or document-style video files into a configured directory using GramJS.

It uses two Telegram identities:

- A bot token, created with BotFather, to receive files you send or forward to the bot.
- A GramJS user session, created with your Telegram API ID/API hash, to download the media from the private chat with the bot.

## Requirements

- Node.js 22.9.0 or newer.
- A Telegram API ID and API hash from <https://my.telegram.org>.
- A Telegram bot token from BotFather.
- Your numeric Telegram user ID for `allowedUserIds`.

## Configuration

Copy the example settings file and fill in your values:

```sh
cp config/settings.example.json config/settings.json
```

`config/settings.json` is ignored by git because it contains secrets.

Settings:

- `telegram.apiId`: Telegram API ID for GramJS.
- `telegram.apiHash`: Telegram API hash for GramJS.
- `telegram.stringSession`: generated by `npm run login`.
- `telegram.botToken`: bot token from BotFather.
- `telegram.botUsername`: bot username without `@`.
- `telegram.allowedUserIds`: Telegram user IDs allowed to trigger downloads.
- `download.directory`: absolute or relative directory where files are saved.
- `download.overwriteExisting`: whether existing filenames may be overwritten.
- `app.logLevel`: `debug`, `info`, `warn`, or `error`.

## Quick Start

```sh
npm install
cp config/settings.example.json config/settings.json
npm run login
npm run dev
```

Send or forward a video to your bot from an allowed Telegram user. The service will save the media to `download.directory`.
